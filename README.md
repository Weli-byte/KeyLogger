# KeyLogger Projesi

Bu proje, Yazılım Mühendisliği dersi kapsamında **eğitim amaçlı** olarak geliştirilmiştir.  
Projenin amacı, klavye hook mantığını ve veri girişinin nasıl yakalandığını öğrenmektir.
